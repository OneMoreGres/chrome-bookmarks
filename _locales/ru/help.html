Поиск

Поиск осуществляется по заголовкам, url и дате добавления. Правила поиска:

  • /<текст> - закладка в любой папке, которая начинается с <текст>
  • /<текст>/ - закладка в папке <текст>
  • #<текст> - закладка содержит тег, который начинается с <текст>
  • #<текст># - закладка содержит тег <текст>
  • ГГГГ-ММ-ДД - закладка, добавленная в указанную дату. Можно задавать различные части даты, но с "-"
  • !<фильтр> - закладка не соответствует фильтру <фильтр> (!#abc# - без тега 'abc')
  • * - все закладки

Внутри фильтров можно использовать:

  • * - любое количество любых символов (t*t соответствует tt, test, t1t, tqwertyt, ...)
  • ? - один любой символ (t?t соответствует t1t, tqt, t2t, ...)
