Search

Search in title, url and add date. Search rules:

  • /<text> - bookmark is in folder which name starts with <text>
  • /<text>/ - bookmark is in folder <text>
  • #<text> - bookmark contains tag that starts with <text>
  • #<text># - bookmark contains tag <text>
  • YYYY-MM-DD - bookmark is added in provided date. Partial date is supported, but must contain "-"
  • !<filter> - bookmark does not match filter <filter> (!#abc# - without tag 'abc')
  • * - all bookmarks

Inside filter following wildcards can be used:

  • * - any amount of any characters (t*t matches tt, test, t1t, tqwertyt, ...)
  • ? - any single character (t?t matches t1t, tqt, t2t, ...)

WARNING! Spaces in tags and folders are not supported

First 10 results can be opened with Ctrl+index hotkey
